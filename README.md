# Better Websockets

Browse, search, filter and export WebSocket streams directly from the Caido sidebar.

<img width="1278" height="464" alt="image" src="https://github.com/user-attachments/assets/c6440410-b58a-4a17-8777-4378ffac1442" />

Mostly vibecoded and in daily use by me.

## Why?

Originally part of [another](https://github.com/v0xyc0n/caido-reqres-exporter) Caido Plugin I made, but as it evolved in functionality it became its own thing.

Major things different from the Caido-Internal websocket functionality:
- One click to filter out Mozilla Push Service requests cluttering up everything
- One click to see the whole history of a stream (not just one message)
- One click to copy or save the history of a stream
- Ability to search streams and within streams
